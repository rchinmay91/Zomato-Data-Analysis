# Zomato Restaurant Data Analysis Using Python

## 📌 Project Overview
This project performs an Exploratory Data Analysis (EDA) on Zomato's restaurant dataset using Python. The goal is to uncover real-world business insights regarding restaurant performance, pricing structures, online delivery patterns, and customer preferences.

## 🛠️ Tech Stack & Tools
* **Platform:** Google Colab / Jupyter Notebook
* **Languages:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn

## 🔍 Key Steps Performed
1. **Data Cleaning:** Handled missing values, dropped duplicate entries, and converted mismatched data types.
2. **Exploratory Data Analysis (EDA):** Analysed correlations between pricing, ratings, and locations.
3. **Data Visualization:** Built distribution plots, bar charts, and heatmaps to reveal hidden dining trends.

## 💡 Top Inferences
* **Delivery Impact:** Restaurants offering online ordering options hold significantly higher average ratings.
* **Price Dynamics:** Most highly-rated restaurants lean toward a moderate-to-high cost-for-two scale, confirming consumers link price with quality.
* **Cuisine Dominance:** North Indian and Fast Food emerge as the highest-volume listed cuisines.
